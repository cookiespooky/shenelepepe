# Быстрый старт

Сделайте шаги ниже и получите рабочий сайт без сложной настройки.

## Шаг 1. Нажмите Use this template на GitHub

Сделайте шаблон своим репозиторием.

![Use this template](/media/use_this_template.webp)

## Шаг 2. Создайте репозиторий из шаблона

После Use this template дайте название вашему репозиторию и сохраните.

![Create repository from template](/media/create-repo.webp)

## Шаг 3. Установите GitHub Desktop

Установите GitHub Desktop и войдите в свой GitHub-аккаунт.

![GitHub Desktop](/media/github_desktop1.webp)

## Шаг 4. Склонируйте репозиторий

Сделайте клон репозитория на компьютер.

![Клонирование репозитория](/media/github_desktop2.webp)

## Шаг 5. Откройте папку как vault в Obsidian

Откройте папку полученного репозитория как хранилище в Obsidian.

![Открытие папки как vault](/media/obsidian_open_as_vault.webp)

## Шаг 6. Установите Obsidian Git plugin

Включите Community plugins в настройках Obsidian и установите плагин Git.

![Включение плагинов](/media/obsidian_plugins_enable.webp)

![Поиск Obsidian Git](/media/obsidian_plugins_git.webp)

![Установка Obsidian Git](/media/obsidian_git_install.webp)

![Включение Obsidian Git](/media/obsidian_git_enable.webp)

## Шаг 7. Commit and Sync — публикация запущена

После Commit and Sync сайт опубликуется в течение минуты. Процесс можно наблюдать в разделе **Actions** в GitHub-репозитории.

Ваш сайт будет доступен по адресу: <имя пользователя>.github.io/<название репозитория>
