# Правила маршрутов и типов

Файл `.np/rules.yaml` задаёт:

- обязательные и опциональные поля frontmatter
- типы страниц и их permalink
- правила резолва ссылок
- коллекции, sitemap и search

Используйте этот файл как основной источник логики модели контента.
