# Конфиг проекта

Файл `.np/config.yaml` управляет:

- базовыми параметрами сайта
- путями к контенту, артефактам и теме
- runtime-режимом URL
- server/cache настройками

`Site.md` и `Interface.md` позволяют задавать пользовательские настройки без ручной правки технического конфига.
