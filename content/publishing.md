# Публикация

Вы вносите изменения → делаете Commit → сайт обновляется автоматически.

## Что делает Commit

- сохраняет изменения в вашем проекте
- отправляет их в GitHub
- запускает автоматическое обновление сайта

## Когда сайт обновляется

Обычно через несколько минут после Commit.

## Как часто обновлять

Обновляйте сайт так часто, как вам удобно:

- после каждой важной правки
- после публикации новой заметки
- после обновления структуры разделов

## Практика без лишних действий

Сделайте правку в заметке, сделайте Commit, и получите новую версию сайта.
