## Почему это удобно

- Публикация прямо из Obsidian
- Уже настроенное SEO
- Быстрая индексация страниц
- Полный контроль над контентом
- Без CMS и админок
- Работает через Git (без серверов)

## Быстрый путь

1. Нажмите “Use this template”
2. Склонируйте через GitHub Desktop
3. Откройте папку в Obsidian
4. Напишите заметку
5. Нажмите Commit → сайт обновится

Подробные шаги: [[start|Быстрый старт]].
