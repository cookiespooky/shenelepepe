# Свойства страницы

Frontmatter — это блок в начале заметки. Он задает свойства страницы: адрес, заголовок, описание и место в навигации.

## Пример frontmatter

```yaml
---
type: article
slug: my-first-note
title: Моя первая заметка
description: Краткое описание страницы для списка и поиска.
order: 30
draft: false
related:
  - start
  - publishing
---
```

## Что значит каждое свойство

- `type` — тип страницы по правилам шаблона. Для обычной страницы используйте `article`.
- `slug` — адрес страницы в URL.  
  Пример: `slug: my-first-note` → `/my-first-note/`.
- `title` — заголовок страницы (виден в контенте, списках и поиске).
- `description` — короткое описание страницы (используется в карточках и метаданных).
- `order` — порядок в списках и навигации. Чем меньше число, тем выше страница.
- `draft` — черновик:
  - `false` — страница публикуется;
  - `true` — страница скрыта из публикации.
- `related` — связанные страницы по `slug`, чтобы добавить полезные переходы.

## Короткое правило

Сначала заполните frontmatter → затем пишите текст заметки.

## Пример связи страниц

Добавьте `related`, чтобы пользователь переходил к следующему шагу, например:
- [[start|Быстрый старт]]
- [[publishing|Публикация]]
